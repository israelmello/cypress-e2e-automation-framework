# Cypress E2E Automation Framework

## Overview

This project demonstrates an end-to-end automation framework built with Cypress, following good automation practices and a scalable architecture.

The main goals of this project are:

- Demonstrate Cypress best practices
- Apply the Page Object Model (POM)
- Generate test data using Faker
- Produce HTML reports with Mochawesome
- Keep the project organized and maintainable

## Technologies

- Cypress
- JavaScript
- FakerJS
- Mochawesome
- Prettier

## Project Structure

```text
cypress
├── e2e
├── factories
├── fixtures
├── pageObjects
└── support
```

## Running the Project

Install dependencies:

```bash
npm install
```

Open Cypress:

```bash
npm run cy:open
```

Run all tests:

```bash
npm run cy:run
```
